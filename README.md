# Trash-Zilla

# Sobre o projeto

**Trash-Zilla** é um projeto de lixeira inteligente criado por alunos do terceiro período de Engenharia de Computação da PUC MG - Campus Coração Eucarístico. Utilizando inteligência artificial, o Trash-Zilla identifica o tipo de lixo inserido pelo usuário e realiza a separação automática, facilitando a reciclagem e promovendo a sustentabilidade de maneira inovadora.

# Tecnologias utilizadas

- MQTT Broker
- MQTT Pannel
- MQTT Dash
- PostgreSQL
- TensorFlow
- C++
